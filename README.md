# Healthcare Clinic Dashboard - Patient & Treatment Analytics

**Technologies:** Power BI, DAX, Data Modeling, Data Visualization

## Project Overview
This project is a data analysis dashboard built in Power BI to monitor and explore patient and treatment data.

## Features
- Interactive dashboard analyzing 100K+ patient records.
- Tracks key KPIs: average age, visit cost, patient count.
- Dynamic slicers and DAX measures for insights on demographics and diagnosis distribution.

## Folder Structure
- `PowerBI_Files/` : Contains the main `.pbix` Power BI file.
- `Data/` : Sample or anonymized dataset used for the dashboard.
- `Documentation/` : Additional notes on metrics, KPIs, and design decisions.
